# Streamlit Tutorials

#### Author : Rahul Bhoyar

Welcome to the Streamlit Tutorials repository! Here, you'll find a collection of tutorials to help you get started with Streamlit, a powerful Python library for building interactive web apps.

## What is Streamlit?

Streamlit is an open-source framework that allows you to create beautiful, interactive web apps directly from your Python scripts. With Streamlit, you can effortlessly turn your data analysis scripts, machine learning models, visualizations, and more into web applications.

## Getting Started

To begin exploring Streamlit, check out the tutorials available in this repository. Each tutorial provides step-by-step instructions and example code to help you understand the basics of using Streamlit for various purposes.

## Repository Structure

- `app1_setting_page_config.py`: Introduction to Streamlit and basic usage.
- `02_building_data_visualizations.md`: Creating interactive data visualizations with Streamlit.
- `03_deploying_machine_learning_models.md`: Deploying machine learning models with Streamlit.
- `04_advanced_features_and_customization.md`: Exploring advanced features and customization options in Streamlit.

Feel free to explore, experiment, and learn at your own pace. Happy Streamlit coding!


## Execution
#### Step 1 : Install the requirements by executing thr following command.
```
pip install -r requirements.txt
```
It is advisable to install these requirements by creating Python virtual environment.

#### Step 2 : Execute the following command to run the streamlit app.

```
streamlit run <file_name.py>
```
